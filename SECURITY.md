# Security Policy

## Supported Versions

Security fixes are currently applied to the latest version on `main`.

| Version | Supported |
| --- | --- |
| Latest (`main`) | Yes |
| Older releases | No |

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Use GitHub Security Advisories to report privately:

- https://github.com/rennerdo30/ollama-web-manager/security/advisories/new

When reporting, include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Suggested mitigation (if known)

We will acknowledge reports as quickly as possible and coordinate a fix and disclosure timeline.
